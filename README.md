# FirebaseProyect

Este es mi primer proyecto utilizando Firebase, el objetivo es praticar sobre esta herramienta y ganar esperiencias en ella.

## Inicio de la app

En esta app hemos utilizado una base de datos creada en `firebase` esto con el fin de crear un `CRUD` y praticar este ejercicio en este servicio.
